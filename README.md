## Project Name & Pitch

Cody

built with Mongo, Express, React and Node.js.

## Installation and Setup Instructions


Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Server:

`npm start`  

To Start Client:

`npm start`

To Visit App:

`localhost:3000`
